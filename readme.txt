Implementation:

Popups are made as a js component that is loaded onto the page
in <div id="popup"> from the corresponding file in the Components folder.

The filter functionality of the account_find_employees.html page is
implemented in scripts/services.js

Slider and menu functions are implemented in scripts/script.js

---------------------------------------------

Site logic that needs to be implemented:

After user registration/authorization from the menu and footer 
you need to remove the Home(index.html) page, and instead of the page 
find_employees.html the user should land on the page 
account_find_employees.html where he can select a category 
and place an order.



---RU---

Реализация:

Попапы сделаны как js-компонент, который подгружается на страницу
в <div id="popup"> из соответствующего файла в папке Components.

Функционал фильтров страницы account_find_employees.html реалзиован
в scripts/services.js

Функции слайдера и меню реализованы в scripts/script.js

---------------------------------------------

Логика сайта, которую нужно реализовать:

После регистрации/авторизации пользователя из меню и футера 
нужно удалить страницу Home(index.html), а вместо страницы 
find_employees.html пользователь должен попадать на страницу 
account_find_employees.html, где он сможет выбрать категорию 
и оформить заказ.
